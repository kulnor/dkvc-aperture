## Input

**Purpose:** Styled text input primitive on `@base-ui/react/input`.
**File:** `src/components/ui/input.tsx`

### Props
Forwards all `@base-ui/react/input` `Input.Props` (`value`, `onValueChange`/`onChange`, `placeholder`, `disabled`, etc.) plus `className`.

### Renders
A single-line input matching the design tokens (border, focus ring, `aria-invalid` destructive ring, disabled opacity, dark-mode fill).
